# Churn-prediction-

Project Overview
This project focuses on analyzing Amazon sales data to identify trends in product pricing, discounts, ratings, and overall product performance. The objective is to perform exploratory data analysis and generate insights that support data-driven business decisions.
________________________________________
Business Problem
E-commerce platforms generate large volumes of sales data. Understanding how pricing, discounts, and customer ratings impact product performance is critical for optimizing sales strategies.
________________________________________
Dataset Description
The dataset contains information related to Amazon products, including:
•	Product name and category
•	Price and discount details
•	Customer ratings and review counts
________________________________________
Tools and Technologies
•	Python
•	Pandas, NumPy
•	Matplotlib, Seaborn
________________________________________
Project Workflow
1.	Imported and explored the dataset
2.	Cleaned data by handling missing values, duplicates, and inconsistent formats
3.	Performed exploratory data analysis to uncover trends and patterns
4.	Created visualizations to support insights and findings
________________________________________
Key Insights
•	Products with higher discounts generally showed increased customer engagement
•	Certain product categories consistently outperformed others in ratings
•	Pricing patterns varied significantly across categories
________________________________________
Conclusion
The analysis provided valuable insights into sales performance and customer behavior. These insights can help businesses improve pricing strategies and product positioning.
